# Nodejs gateway

## 环境

1. nodejs
2. npm or yarn
3. express
4. etc...

## 实现功能

1. 发布 RESTful API 供调用。
2. 通过 gRPC 调用后台业务。
3. 实现接口权限控制 ( Password, OAuth, JWT )。
4. 实现 Swagger 集成。
5. 集群化部署。
